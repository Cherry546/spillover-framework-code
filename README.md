\# Code for spillover framework manuscript



This repository contains Python scripts used to generate illustrative figures in the manuscript.



\## Requirements



Python 3.11 or later.



Install dependencies:



pip install -r requirements.txt



\## Reproduce figures



Run the following commands from the repository root directory:



python scripts/fig2\_spatial\_driver\_surface.py

python scripts/fig3\_lyme\_acquisition\_component.py

python scripts/fig5\_movement\_overlap.py



Generated figures are saved in outputs/figures/.
